# คู่มือเริ่มต้นใช้งานระบบ Timesheet (ฉบับย่อ)

## 1. เข้าสู่ระบบ

1. เปิดเว็บไซต์ระบบ Timesheet ขององค์กร
2. กรอก **อีเมล** และ **รหัสผ่าน** ในช่องที่กำหนด
3. กดปุ่มสีเขียว **"Sign In"** — หรือกดปุ่ม **"Sign in with Microsoft"** เพื่อเข้าด้วยบัญชี Microsoft ขององค์กร

> ลืมรหัสผ่าน? กดลิงก์ **"Forgot password?"** ด้านล่าง แล้วทำตามอีเมลที่ระบบส่งให้

## 2. กรอกเวลาทำงาน

1. กดปุ่ม **"Open Timesheet →"** ที่หน้าหลัก (หรือเลือกเมนู Timesheet จากแถบด้านซ้าย)
2. กดปุ่ม **"+ Add Charge Code"** ด้านล่างซ้ายของตาราง แล้วเลือกรหัสงานที่ต้องการ
3. **กดที่ช่องว่าง** ในตาราง (ตรงจุดตัดระหว่างรหัสงานกับวัน) แล้วพิมพ์จำนวนชั่วโมง เช่น 8.00
4. ดูแถว **"Daily Total"** ให้แต่ละวันครบ **8 ชั่วโมง** — ถ้ายังไม่ครบจะเห็นตัวเลขสีแดงที่แถว **"Variance"**

> ระบบบันทึกให้อัตโนมัติทุก 30 วินาที ไม่ต้องกลัวข้อมูลหาย — หรือกดปุ่ม **"Save Draft"** เพื่อบันทึกเอง

## 3. ส่งใบบันทึกเวลา

1. ตรวจสอบว่าทุกวันทำงาน (จ.–ศ.) มียอดรวมครบ 8 ชั่วโมง
2. กดปุ่มสีเขียว **"Submit →"** ที่มุมขวาล่าง — สถานะจะเปลี่ยนเป็น **"Submitted"** และหัวหน้าจะได้รับแจ้งเตือนทันที

## 4. ขอลาพักร้อน

1. กดเมนู **ปฏิทิน** จากแถบด้านซ้าย
2. กดปุ่มสีเขียว **"+ Request Vacation"** ที่มุมขวาบนของปฏิทิน
3. เลือก **วันเริ่ม**, **วันสิ้นสุด**, และประเภทการลา (เต็มวัน / ครึ่งวันเช้า / ครึ่งวันบ่าย) แล้วกดส่ง

> คำขอลาจะถูกส่งไปให้หัวหน้าอนุมัติ — ดูสถานะได้ที่ส่วน **"My Vacation Requests"** ด้านล่างปฏิทิน

## 5. ติดต่อเมื่อมีปัญหา

| ปัญหา | ติดต่อ |
|-------|--------|
| ลืมรหัสผ่าน / เข้าระบบไม่ได้ | กด **"Forgot password?"** หรือติดต่อฝ่าย IT |
| ไม่เห็นรหัสงานที่ต้องการ | กดปุ่ม **"+ Request New CC"** ในหน้า Timesheet |
| ใบบันทึกเวลาถูกตีกลับ | ตรวจสอบเหตุผลในระบบ แล้วแก้ไขและส่งใหม่ |
| เลยกำหนดส่ง (แถบสีแดง) | ติดต่อหัวหน้างานโดยตรง |
| ปัญหาอื่น ๆ | กดปุ่มแชท **มุมขวาล่าง** เพื่อคุยกับผู้ช่วยอัตโนมัติ หรือติดต่อฝ่าย IT |
